# This is a reference list of 30 blog post ideas, used manually when
# either you (Malek) or the client decide it's time to publish a post.
#
# How to use it:
#   - Pick any topic below.
#   - Run:  npm run draft "<that exact topic>"
#     (or: npm run draft:queue   to auto-pick the first non-# line)
#   - The drafter writes src/content/blog/<slug>.mdx with `draft: true`.
#   - Review/edit, remove the `draft: true` line, commit, push.
#   - Vercel auto-deploys, post is live.
#
# After using a topic, prepend "# " to that line so you don't reuse it.
# Add new ideas by appending to the end of the file.
#
# NOT automated: there's no cron job — drafts only happen when someone
# manually runs the script. API costs (~$0.05–$0.10 per post via Claude)
# go to whoever's ANTHROPIC_API_KEY is in their local .env.local.
